1. for you to successfully run the app you need vs code and mysql installed on the host computer
2. start the mysql workbench and import data in "./database/shumba_moneydb.sql"
3.open the server folder and the index.js file
4. in the index.js file change the db const password and username
5.open the server folder in the intergrated terminal of vscode and execute the following command "npm start devStart"
6.open the client folder in the intergrated terminal and execute the command "npm start"
